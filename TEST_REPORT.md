## 3. 非功能测试

### 3.1 流式响应时间

| 测试场景 | 首次token延迟 | 完整响应时间 |
|---------|-------------|------------|
| 普通对话 | 1-3秒 | 5-15秒 |
| 资源生成（多智能体协作） | 3-8秒/智能体 | 20-60秒（含5个智能体） |
| 智能辅导 | 2-5秒 | 10-30秒 |
| 效果评估 | 2-5秒 | 10-20秒 |

### 3.2 并发性测试

| 并发连接数 | 表现 |
|-----------|------|
| 1 | 正常 |
| 3 | 正常，响应时间略有增加 |
| 5+ | 可能出现响应延迟，建议生产环境使用异步Worker |

### 3.3 错误处理

| 异常场景 | 处理方式 |
|---------|---------|
| API Key无效 | 启动时抛出ValueError |
| API请求超时（120s） | requests timeout自动抛出 |
| JSON解析失败 | extract_json 3级策略+错误修复 |
| SSE连接断开 | request.is_disconnected() 检测退出 |
| 课程知识库缺失 | 使用默认配置，不阻断运行 |
| 模型返回空内容 | 空字符串处理，智能体继续执行 |
